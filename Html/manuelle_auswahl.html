Manuelle Auswahl

Die Manuelle Auswahl funktioniert z.Zt. nur für ausgedruckte Fragebögen. Sie bietet sich an wenn:

  • es strittige Fragen gibt
  • wenn ein leichter/schwerer Fragebogen erstellt werden soll
  • wenn für einen Übungsabend ein Fragebogen zu einem Themenschwerpunkt benötigt wird

Sie kann rein manuell erfolgen, aus einer Suche entstehen oder einer Kombination aus beiden Möglichkeiten.

Der "Manuelle Auswahl"-Dialog

Achtung: Eine noch nicht gespeicherte Manuelle Auswahl wird OHNE Rückfrage überschrieben wenn einer der Button oder gedrückt wird.

Eine Auswahl laden

Links oben erscheinen die Dateinamen der bisher gespeicherten Manuellen Auswahlen. Mit einem Doppelklick auf einen Dateinamen wird diese Manuelle Auswahl geladen und im "Manuelle Auswahl"-Formular angezeigt. Alternativ kann man auch den Dateinamen mit der Maus auswählen und auf den Button klicken.

Eine Auswahl löschen

Die zu löschende Datei mit der Maus auswählen und auf den Button klicken. Die folgende Rückfrage mit beantworten wenn die Datei wirklich gelöscht werden soll.

Eine neue, leere Auswahl erstellen

Der unter Button erstellt eine LEERE Auswahl und wechselt in das Formular "Manuelle Auswahl".

Zur aktuellen Manuellen Auswahl wechseln

Der unter Button zeigt die AKTUELLE Auswahl im "Manuelle Auswahl"-Formular an.

Das "Manuelle Auswahl"-Formular

Überblick

Ganz oben steht der Name der Auswahl. Direkt darunter die gesamte Anzahl der Fragen.

Oben in der Drop-Down Liste steht das aktuelle Kapitel mit der Anzahl der aus diesem Kapitel ausgewählten Fragen. Die Text-Farbe in der Drop-Down Liste bedeutet:

  • rot: KEINE Fragen aus diesem Kapitel
  • blau: teilweise Fragen aus diesem Kapitel
  • grün: ALLE Fragen aus diesem Kapitel

Um zu einem anderen Kapitel zu wechseln ruft man dieses einfach aus der Drop Down Liste auf.

Darunter findet man folgende Buttons zur Fragen-Auswahl:

  • -> markiert ALLE Fragen aus diesem Kapitel
  • -> demarkiert ALLE Fragen aus diesem Kapitel
  • -> dreht die aktuelle Auswahl um

Dann folgen die einzelnen Fragen. Die markierten Fragen sind blau und fett. Es wird nur die Frage gezeigt. Ein Klick auf das rote Kreuz am rechten Rand zeigt auch die Antworten an bzw. blendet sie wieder aus.

Speichern und Komplette Auswahlen

Für eine gespeicherten Auswahl steht der - Button zur Verfügung.
