System-Voraussetzungen

Betriebssystem

  • Das Programm läuft nur auf Windows Rechnern ab Windows XP
  • Das Programm benötigt den Internet-Explorer (ab 6.0)

Fragenkataloge

Ich kann aus rechtlichen Gründen keine Fragen-Kataloge für die neue Feuerwehrleistungsübung der Hessischen Landesfeuerwehrschule mitliefern. Das heißt die Fragen müssen selber abgeschrieben werden oder man kennt jemanden der das schon gemacht hat.

Javascript

Der Internet-Explorer muss JavaScript erlauben. Du erkennst es daran, das immer nur leere, weiße Seiten angezeigt werden.

JavaScript im Internet-Explorer aktivieren

  1. Öffne Deinen Internet-Explorer
  2. Gehe im Menü unter Extras Internetoptionen und wähle dann den Reiter Sicherheit.
  3. Wähle die Zone Internet und klicke auf den Button .
  4. In dem neu erscheinenden Fenster findest Du ungefähr in der Mitte die Einstellungen für Scripting (Siehe Bild rechts).
  5. Markiere unter Active Scripting die Option Aktivieren.
  6. Klicke auf um die Änderungen zu übernehmen.
  7. Es folgt noch einmal eine Sicherheitsabfrage die Du dann mit   beantworten musst.
  8. Klicke zum Abschluss auf oder um das Fenster Internetoptionen zu schließen.
  9. Zum Abschluß kannst Du den InternetExplorer wieder schließen.

Zurück im Fragebogen-Programm wähle im Menü unter Datei den Punkt Kapitel aktualisieren aus.
