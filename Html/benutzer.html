Benutzer-Verwaltung

Vorwort

Bisher wurden alle Ergebnisse im Verzeichnis Benutzer/Fremder gespeichert. Bei der Weitergabe des Programms durfte man das Verzeichnis Benutzer nicht mit kopieren (sollte man auch jetzt nicht machen).

Wenn der Empfänger diese Dateien in seine eigenen Fragebogen-Ordner kopiert hat, wurde sein Ausbildungsstand mit dem des Absenders überschrieben. Dann war seine (und meine) ganze Arbeit umsonst.

Jetzt wird der Name des Benutzers abgefragt als Ordner-Name im Verzeichnis Benutzer verwendet. Gibt man alle Dateien weiter, bekommt der Empfänger einen zusätzlichen Benutzer, den er später löschen kann.

Hier gibt es allgemeine Hinweise zur Programm-Weitergabe

Benutzer-Verwaltung

Unter dem Menüpunkt Benutzer stehen 3 Untermenüs zur Verfügung:

  • Eigenen Namen ändern

    Sollte in den seltesten Fällen notwendig sein

  • Neuen Benutzer einrichten

    Innerhalb der Familie, zu Übungszwecken, maximal 9 Benutzer

    Wenn vorhanden, können bestimmte Einstellungen vom aktuellen Benutzer übernommen werden.

  • Benutzer löschen
    siehe Vorwort, Test-Benutzer.

    Achtung: gelöschte Benutzer sind für immer gelöscht. Man kann sie nicht wieder aus dem Papierkorb zurück holen.

    Sicherer ist es, das Verzeichnis mit dem Benutzernamen mittels Windows-Explorer aus dem Verzeichnis Benutzer an eine andere Stelle zu kopieren.

Bereits vorhanden Benutzer-Namen können nicht noch einmal vergeben werden. Das Eingabe-Feld zeigt dann den Namen in Rot. Wenn der Cursor darauf gestellt wird, erscheint eine kurze Begründung.

Benutzer-Liste / Benutzer-Wechsel

Unterhalb des Menüpunkts Benutzer-Verwaltung wird pro Unterordner im Verzeichnis Benutzer ein Eintrag erstellt. Der beim Start ausgewählte Benutzer wird mit einem Häckchen angezeigt und ist deaktiviert.

Gibt es nur einen Benutzer ist die Geschichte hier zu Ende. Bei mehreren Benutzern führt ein Klick auf einen anderen Namen für direkt zur Anzeige dieses Benutzers

Alle Einstellungen und Wissenstände werden entsprechend neu geladen. Der Spitzname wird rechts oben bei Ausbildungsstand angezeigt.

Probleme bei mehrenen Benutzern

Es muss beim Start immer der zu startende Benutzer ausgewählt werden. (Lösung für dieses Problem: siehe folgenden Absatz)

Alle Benutzer teilen sich das Verzeichnis Fragen. D. h. Verschiebungen zwischen aktuellen Kapiteln und archivierten Kapiteln mittels Kapitel-Explorer oder manuell wirken sich immer auf alle Benutzer aus.

Die Gefahr, sich selber mal aus Versehen selber zu löschen.

Starten mit einer Verknüpfung

Wenn auf Dauer mehrere Personen auf dem gleichen Rechner arbeiten müssen empfiehlt sich die Einrichtung einer Verknüpfung für jede Person.

  • Wenn man im Windows-Explorer mit dem Cursor auf die Datei fragebogen.exe zeigt, kann man durch einen Klick auf die rechte Maustaste aus dem dann erscheinenden Kontext-Menü den Punkt Verknüpfung erstellen auswählen.
  • Der Computer erzeugt sofort eine neue Datei mit dem Namen fragebogen.exe - Verknüpfung
  • Jetzt klickt man mit der rechten Maustaste auf diese neue fragebogen.exe - Verknüpfung und wählt aus dem Kontext-Menü den Punkt Eigenschaften an.
  • Im Reiter Verknüpfung steht im Eingabefeld Ziel der gesamte Datei-Pfad in Anführungszeichen. Ganz am Ende gibt man nun zusätzlich ein Leerzeichen und dann - wieder in Anführungszeichen - "name=benutzername" ein. (Klein- und Großschreibung beachten)
    (Bei mir sieht das dann so aus: "C:\Users\Manfred\Documents\Fragebogen\fragebogen.exe" "name=Manfred Altenhein")
  • Zum Speichern klickt man noch auf den OK-Button
  • Mit Doppelklick auf diese Datei startet dann das Fragebogen-Programm mit dem Benutzer-Namen OHNE Rückfrage
  • Diese verknüpfte Datei kann auch einfach auf den Desktop gezogen und beliebig benannt werden

Bei Bedarf ist dieser Vorgang dann für jede Person zu wiederholen

Alternativen

Um einen weiteren Benutzer einzurichten, kannst Du das Programm einfach in ein anderen Verzeichnis oder auf einen (anderen) USB-Stick kopieren.
